# KSRCE Results Portal

A simple website that displays the KSRCE examination results portal.

## How to Use

Open the website locally in your browser:

```powershell
cd 'C:\Users\durai\OneDrive\Documents\projects\OPERATION\cgpa_calculator'
Start-Process .\index.html
```

Or use:
```powershell
ii .\index.html
```

## Features

- Clean, modern interface
- Embedded KSRCE results portal via iframe
- Fallback link if iframe is blocked
- Responsive design for mobile and desktop

## Note

If the iframe doesn't display the external site (due to X-Frame-Options restrictions), use the fallback link to open the portal in a new tab.
